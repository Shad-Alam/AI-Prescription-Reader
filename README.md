# AI-Prescription-Reader 

## :point_right: Overview
To transform confusing medical prescriptions into clear, simple instructions that anyone can understand instantly.

## :point_right: Key Features
- **Camera Scan** - Live prescription scanning
- **ML Kit OCR** - Offline text recognition
- **Medicine Detection** - Finds common medicines
- **No Internet** - Works completely offline
- **Simple UI** - Easy to use interface

## :point_right: Tech Stack
- **Language**: Java
- **Min SDK**: Android 5.0 (API 21)
- **Libraries**:
  - CameraX (Camera operations)
  - ML Kit (Text recognition)

## :point_right: Installation
1. Clone the repository
2. Open in Android Studio
3. Build the project
4. Run on device/emulator

## :point_right: Usage
1. Open the app
2. Tap "Camera Scan"
3. Position prescription in frame
4. Capture image
5. View extracted text and analysis
6. Copy or share results

## :point_right: Project Structure
- MainActivity.java - Home screen
- ScannerActivity.java - Camera interface
- ResultActivity.java - Results display
- OCRManager.java - Text recognition
- MedicineAnalyzer.java - Medicine analysis

## :point_right: Requirements
- Android 5.0+ (API 21)
- Camera permission
